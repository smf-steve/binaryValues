COMP122: Binary Mapping and Values Library

This webpage has been designed to allow students to execute various methods to manipulate binary numbers. The intent is for students to open the console to execute this routines to help them in various assignments.

The Javascript program that is loaded via the web page is NOT intended to be included in larger programs.

Opening the Console

To open the console on various browsers, enter the following command sequence:
  • Brave: Option + ⌘ + J (macOS), or Shift + CTRL + J (Windows/Linux)
  • Chrome: Option + ⌘ + J (macOS), or Shift + CTRL + J (Windows/Linux)
  • FireFox: Option + ⌥ + C (macOS), or Shift + CTRL + J (Windows/Linux)
  • Safari: Option + ⌥ + C (macOS)
Note: You might have to enable the "developer" mode on your browser.
Note: We recommend that you side by side viewing mode.

Number Representation:

All numbers are represented as a string. This string begins with "base #". As such, the decimal number 42, can be represented as:
  • "2# 101010"
  • "8# 52"
  • "10# 42"
  • "16# 2A"

Also note that space and superfluous zeros may be placed with the string for visual clarity. As such the decimal number 2,776 can be represented as:

  • "2# 0000 1010 1100 1110"
  • "8# 5 316"
  • "10# 2 776"
  • "16# 0A CE"

Supported API:

Conversions:

  • Number.toBin()
  • Number.toOctal()
  • Number.toHex()
  • Number.toDec()

Formating:

  • Number = Number.shrink(): removes superfluous zeros
  • Number = Number.expand(N): adds superflous zeros to ensure the length is N
  • Number = Number.nibbles(): groups of 4-bit quanities
  • Number = Number.bytes(): groups of 8-bit quanities
  • Number = Number.halfs(): groups of 16-bit quanities
  • Number = Number.words(): groups of 32-bit quanities
  • Number = Number.doubles(): groups of 64-bit quanities

  • Number = Number.separate(N): each group is of size N
  • Number = Number.squeeze(): removes superflous spaces

Mappings: Character <-> Binary

  • 'char' = Number.encodeBase64()
  • Number = 'char'.decodeBase64()
  • Number = 'char'.encodeASCII()
  • 'char' = "Number".decodeASCII()
  • Number = 'char'.encodeUTF8()
  • 'char' = Number.decodeUTF()

Chopping and Join:

  • Number[] = numberChop(N)
    Creates an array of Numbers by chopping the orginal number into groups of N digits.
    If N is undefined, then number is chopped at each space.
    By default the number is chopped starting from the right.
  • Number =(Number[]).numberJoin(base=2)
  • Number =(Number[]).splice(Number, ba) # follows the pattter of Array.splice -- Number[] = Number.numberChop() -- Number[] = (Number[]).splice(start, delCount, item) // in place -- Number = Array.numberJoin()
